# 期末复习助手 - 设计文档

## 项目概述

**做什么：** 一个期末复习提纲生成网站，上传PPT/Word文档后，AI自动生成带有重点标注的知识点提纲，并支持题库刷题功能。

**为什么：** 帮助大学生快速从大量复习资料中提取核心知识点，特别是常考易考点，提高复习效率。

**给谁用：** 同学、室友之间小范围分享使用。

---

## 核心功能

### 1. 复习资料处理

| 功能 | 说明 |
|-----|------|
| 文件上传 | 支持PPT、Word文档（.pptx, .docx） |
| 文档解析 | 提取文本内容，识别标题层级 |
| AI提纲生成 | 生成带重点标注的知识点提纲 |
| 重点标注 | 🔴必考重点、⭐常考易考点、📌了解即可 |
| PDF导出 | 可下载PDF方便打印 |

### 2. 题库刷题

| 功能 | 说明 |
|-----|------|
| 题库上传 | 支持JSON、Excel格式上传 |
| AI自动出题 | 从复习资料中AI生成题目 |
| 多种题型 | 单选、多选、判断 |
| 章节练习 | 按章节分类刷题 |
| 错题本 | 自动记录错题，可重做 |
| 收藏功能 | 收藏重要题目 |
| 统计分析 | 正确率、答题数、章节掌握度 |

### 3. 分享功能

| 功能 | 说明 |
|-----|------|
| 链接分享 | 生成分享链接，同学可直接访问 |
| 有效期设置 | 7天或30天可选 |
| 无需登录 | 打开即用，无需注册 |

## 核心流程

### 主流程

```
用户上传文件 → 解析文档 → 提取文本 → AI生成提纲 → 展示结果 → 分享/导出
```

### 题库流程

```
上传题库文件 → 解析题目 → 存储题库 → 刷题练习 → 记录错题 → 统计分析
```

### AI出题流程

```
选择提纲 → AI生成题目 → 存储题库 → 刷题练习
```

```javascript
{
  "id": 1,
  "type": "single",  // single=单选, multi=多选, judge=判断
  "question": "题目内容",
  "options": [
    {"key": "A", "value": "选项内容"},
    {"key": "B", "value": "选项内容"},

  "explanation": "解析说明"
```

## API设计

| 方法 | 路径 | 功能 |
|-----|------|------|
| POST | `/api/upload` | 上传文档 |
| GET | `/api/outline/:id` | 获取提纲 |
| POST | `/api/generate` | AI生成提纲 |
| POST | `/api/quiz/generate` | AI生成题目 |
| POST | `/api/quiz/upload` | 上传题库 |
| GET | `/api/quiz/list` | 获取题库列表 |
| GET | `/api/quiz/:id` | 获取题目 |
| POST | `/api/share` | 生成分享链接 |
| GET | `/api/share/:id` | 获取分享内容 |

## 数据存储

### 使用 Vercel KV (Redis)

| 数据类型 | Key格式 | TTL | 说明 |
|---------|---------|-----|------|
| 文件 | `file:{file_id}` | 7天 | 节省存储空间 |
| 提纲 | `outline:{outline_id}` | 30天 | 可能重复查看 |
| 题库 | `quiz:{quiz_id}` | 30天 | 可能重复刷题 |
| 分享 | `share:{share_id}` | 7-30天 | 根据用户选择 |
| 用户记录 | `user:{ip}` | 30天 | 保留进度 |

## AI提示词设计

### 提纲生成

- 温度：0.3（低温，确保准确性）
- 输出：JSON格式，包含章节、知识点、重要性标注
- 特点：考前冲刺型，标注重点和常考点

### 题目生成

- 温度：0.7（中温，增加题目多样性）
- 输出：JSON格式，包含题目、选项、答案、解析
- 特点：覆盖知识点，选项有迷惑性

## 前端界面

### 页面结构

1. **首页** - 功能入口（上传资料、我的题库、刷题练习）
2. **上传页面** - 文件上传，处理进度
3. **提纲展示页面** - 知识点提纲，重点标注，导出PDF
4. **题库页面** - 上传题库，AI生成题库
5. **刷题页面** - 答题界面，进度显示，结果统计
6. **分享页面** - 查看分享的提纲或题库

### 设计风格

- 移动端优先，响应式设计
- 蓝色主题（#4A90D9）
- 卡片式布局
- 清晰的层级结构

## 风险与对策

| 风险 | 对策 |
|-----|------|
| AI生成质量不稳定 | 优化提示词，多次重试，人工校验 |
| 文档解析失败 | 支持多种格式，错误提示清晰 |
| API调用超时 | 设置超时时间，异步处理 |
| 存储空间不足 | 自动清理过期数据 |
